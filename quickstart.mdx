## Pré-requisitos

<CardGroup cols={3}>
  <Card title="Conta Lineup" icon="user">
    Entre em <a href="https://lnup.pro/auth">lnup.pro/auth</a> com Discord.
  </Card>
  <Card title="Inbox criada" icon="inbox">
    Crie ou selecione uma Inbox no dashboard.
  </Card>
  <Card title="Token de API" icon="key">
    Gere um token <code>lup_...</code> em Conta → Configurações → API externa.
  </Card>
</CardGroup>

<Warning>
  Nunca coloque o token <code>lup_</code> no frontend do seu produto. Ele deve
  viver apenas no seu backend ou em um cofre de segredos.
</Warning>

## 1. Liste suas Inboxes

<CodeGroup>

```bash cURL
curl "https://api.lnup.pro/v1/inboxes" \
  -H "Authorization: Bearer lup_seu_token"
```

```ts Node.js
const response = await fetch("https://api.lnup.pro/v1/inboxes", {
  headers: {
    Authorization: `Bearer ${process.env.LINEUP_API_TOKEN}`,
  },
});

const inboxes = await response.json();
```

</CodeGroup>

## 2. Crie o ticket

Todos os campos de cliente são opcionais. Envie apenas o que seu produto já
tem com segurança.

<CodeGroup>

```bash cURL
curl -X POST "https://api.lnup.pro/v1/inboxes/inbox_123/tickets" \
  -H "Authorization: Bearer lup_seu_token" \
  -H "Content-Type: application/json" \
  -d '{
    "customerId": "cliente_123",
    "customerName": "Maria",
    "customerEmail": "maria@exemplo.com",
    "subject": "Dúvida sobre cobrança",
    "category": "Financeiro",
    "reason": "Preciso entender uma cobrança recente.",
    "metadata": {
      "orderId": "pedido_456"
    }
  }'
```

```ts Node.js
const response = await fetch(
  "https://api.lnup.pro/v1/inboxes/inbox_123/tickets",
  {
    method: "POST",
    headers: {
      Authorization: `Bearer ${process.env.LINEUP_API_TOKEN}`,
      "Content-Type": "application/json",
    },
    body: JSON.stringify({
      customerId: "cliente_123",
      customerName: "Maria",
      customerEmail: "maria@exemplo.com",
      subject: "Dúvida sobre cobrança",
      category: "Financeiro",
      reason: "Preciso entender uma cobrança recente.",
      metadata: {
        orderId: "pedido_456",
      },
    }),
  },
);

const ticket = await response.json();
```

</CodeGroup>

## 3. Entregue o link ao cliente

A resposta inclui o ticket, o link público e um token temporário de acesso.

```json
{
  "ticket": {
    "_id": "6a2dac84fc78ffca71f63cc0",
    "workspaceType": "inbox",
    "title": "Dúvida sobre cobrança"
  },
  "chatUrl": "https://go.lnup.pro/chat/6a2dac84fc78ffca71f63cc0?access=tok_...",
  "customerAccessToken": "tok_...",
  "accessExpiresAt": "2026-06-21T12:00:00.000Z"
}
```

<Tip>
  Envie <code>chatUrl</code> por e-mail, WhatsApp, painel interno ou redirecione
  o cliente para esse endereço. Depois que o visitante entra, a Lineup salva a
  sessão dele por cookie seguro.
</Tip>

## 4. Responda pela API ou dashboard

```bash
curl -X POST "https://api.lnup.pro/v1/inboxes/inbox_123/tickets/6a2dac84fc78ffca71f63cc0/messages" \
  -H "Authorization: Bearer lup_seu_token" \
  -H "Content-Type: application/json" \
  -d '{
    "authorRole": "agent",
    "authorName": "Equipe Lineup",
    "content": "Oi, Maria. Vou verificar isso agora."
  }'
```

<Info>
  Mensagens enviadas pela API aparecem em tempo real para quem está com o chat
  aberto.
</Info>
