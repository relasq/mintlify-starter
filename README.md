# Lineup Docs

Documentação Mintlify da Lineup.

## Rodar localmente

```bash
npx mintlify dev --path docs
```

Se você abrir o Mintlify usando a pasta `frontend/docs` como raiz do projeto,
use apenas:

```bash
npx mintlify dev
```

## Estrutura

- `docs.json`: configuração do Mintlify.
- `index.mdx`: introdução.
- `concepts/`: conceitos de produto.
- `guides/`: guias práticos.
- `api-reference/openapi.yaml`: referência da API pública.

## Domínio sugerido

Configure o projeto Mintlify em `docs.lnup.pro`.
